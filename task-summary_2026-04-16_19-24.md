## 目标
界面打开时默认还原上次关闭时的状态（位置、尺寸、最大化）。

## 修复内容

### 1. 添加窗口状态持久化
**文件**: `gui/main_window.py`

- 从 PySide6.QtCore 导入 `QPoint`, `QSettings`
- `__init__` 中创建 `self._settings = QSettings("VideoSubtitleApp", "MainWindow")`
- 窗口初始化后调用 `_restore_window_state()` 恢复保存的窗口几何
- 重写 `closeEvent` 在关闭窗口前调用 `_save_window_state()` 保存状态

### 2. 实现细节
```python
def _restore_window_state(self) -> None:
    geo = self._settings.value("geometry")
    if geo:
        self.restoreGeometry(geo)
    state = self._settings.value("windowState")
    if state:
        self.restoreState(state)

def _save_window_state(self) -> None:
    self._settings.setValue("geometry", self.saveGeometry())
    self._settings.setValue("windowState", self.saveState())
```

使用 Qt 原生 QSettings 自动持久化到系统注册表（Windows），保存位置、尺寸、最大化状态、分栏宽度等。
